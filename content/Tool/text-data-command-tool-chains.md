## Display the first/last few lines 'head' and 'tail' ##

```
head -n 5 demo.txt
tail -n 5 demo.txt
```

## Display the custom line ##

```
sed -n '9,10p' demo.txt
```


## Create mutiple dictionaries from 001 to 100 ##

```
mkdir {001..100}
```
